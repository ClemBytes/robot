\doxysubsection{Detailed Description}
A dynamically allocated string structure. 

This structure represents a resizable string buffer, tracking both the allocated and used sizes.


\begin{DoxyItemize}
\item start\+: pointer to the beginning of the character buffer.
\item available\+\_\+size\+: total allocated size in bytes, including already written characters and the final \textquotesingle{}\textbackslash{}0\textquotesingle{}.
\item used\+\_\+size\+: number of bytes currently used in the buffer, including the final \textquotesingle{}\textbackslash{}0\textquotesingle{}. 
\end{DoxyItemize}
